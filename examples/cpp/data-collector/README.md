# Data Collector

## Configuration
- RBY1
  - Model: A
  - Camera:
    - D405 x 2 (End-effector)
    - D435 (Head)
- Master Device

## Prerequisites

1. [Realsense](https://github.com/IntelRealSense/librealsense)
2. [HDF5](https://support.hdfgroup.org/downloads/hdf5/hdf5_1_14_5.html)
    - [HighFive](https://github.com/BlueBrain/HighFive)
3. OpenCV
4. ZeroMQ
    - [libzmq](https://github.com/zeromq/libzmq)
    - [cppzmq](https://github.com/zeromq/cppzmq)
5. JSON
    - [nlohmann_json](https://github.com/nlohmann/json)
